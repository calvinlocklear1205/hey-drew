HeyDrew Screen Gallery

Each link opens a single screen from the prototype, ready for the html.to.design Figma plugin to capture as editable layers.

  1. Run npx serve . from the hey-drew folder
  2. Open this page at http://localhost:3000/screens.html
  3. In Figma, open the html.to.design plugin
  4. Click any link below — it opens the screen in a new tab
  5. Copy that tab's URL into the plugin and import
  6. Repeat for each screen you need
